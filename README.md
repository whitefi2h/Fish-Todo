🚀 Fish Todo: A Smart, Evolving Priority To-Do List

Fish Todo 是一款专为高效执行者设计的轻量级待办事项工具。它不仅仅是一个清单，能够根据任务的拖延情况自动调整优先级，确保重要的事永远不会被淹没。
Fish Todo is a lightweight to-do list tool designed specifically for high-efficiency achievers. It is more than just a checklist; it automatically adjusts task priorities based on how long they have been pending, ensuring that important items never get buried.

✨ 核心特性
✨ Core Features

📈 动态优先级演进 (Time-Shift)：系统会根据任务创建后的停留时间，自动提升其紧急程度：自动升级： 设定天数后，任务会自动从 Waiting 转为 Urgent，最终变为 Overtime，告别拖延： 视觉上的颜色预警（如红色的 Overtime 标签）会不断提醒你处理积压任务。
📈 Dynamic Priority Evolution (Time-Shift): The system automatically increases a task's urgency level based on how long it has remained pending since creation: Automatic Escalation: After a set number of days, tasks automatically transition from "Waiting" to "Urgent," and eventually to "Overtime." Beat Procrastination: Visual color-coded warnings (such as a red "Overtime" tag) serve as constant reminders to tackle your backlog.

🖱️ 交互式任务管理右键编辑：无需繁琐点击，直接在任务卡片上点击鼠标右键即可弹出详细编辑界面，快速补充科研备注或步骤。自动展开详情： 在添加任务时可设置自动打开详情页，确保信息录入完整。
🖱️ Interactive Task Management — Right-Click Editing: No need for tedious clicking; simply right-click directly on a task card to instantly open the detailed editing interface, allowing you to quickly add research notes or steps. Auto-Expand Details: When adding a new task, you can configure the app to automatically open the details page, ensuring that all relevant information is fully captured.

🎨 高度自定义外观主题切换： 支持 Dark（深色）与 Light（浅色）模式。背景透视： 可调节背景不透明度（Background Opacity），让应用与你的桌面背景完美融合。色彩定义： 支持通过十六进制代码（如 #A9D0ED）自定义主题色。
🎨 Highly Customizable Appearance — Theme Switching: Supports both Dark and Light modes. Background Transparency: Adjust the "Background Opacity" setting to seamlessly blend the application window with your desktop wallpaper. Custom Color Schemes: Define your own theme colors using hexadecimal codes (e.g., #A9D0ED).

🛠️ 系统级集成开机自启：确保每天开启电脑的第一眼就能看到今日进度。窗口置顶： 开启 Always on top，在查阅文献或运行代码时，任务清单始终可见。
🛠️ System-Level Integration — Launch at Startup: Ensure that your daily progress is the very first thing you see when you turn on your computer each morning. "Always on Top" Window: Enable this feature to keep your task list visible at all times—perfect for multitasking while reviewing research papers or running code.

⚙️ 如何配置优先级规则：在 Settings 弹窗中，你可以精准控制任务的“进化速度”：Waiting → Urgent (Days): 设置任务存放多少天后变为“紧急”。Urgent → Overtime (Days): 设置紧急任务在多少天后被标记为“超时”。右键点击卡片： 呼出详情编辑。顶部输入框： 快速添加新事项并分配初始优先级（Urgent / Waiting / Not Yet）。
⚙️ How to Configure Priority Rules: In the "Settings" window, you can precisely control the "evolution speed" of your tasks: Waiting → Urgent (Days): Set the number of days a task remains in "Waiting" status before automatically escalating to "Urgent." Urgent → Overtime (Days): Set the number of days an "Urgent" task remains pending before being flagged as "Overtime."
Right-Click on a Card: Open the detailed editing interface. Top Input Bar: Quickly add new items and assign an initial priority level (Urgent / Waiting / Not Yet).

🛠️ Installation & Usage: Visit the "Releases" page. Download the appropriate installation package for your operating system (e.g., `Computer 64-bit Installer.exe`). Run and start managing your complex research projects!
